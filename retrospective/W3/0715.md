# 2026-07-15 회고

## Keep
- 데이터 표준화의 필요성과 표준화가 데이터 품질 및 일관성에 미치는 영향을 학습했다.
- 데이터 모델링 실습을 통해 요구사항을 분석하고 엔티티, 관계, 속성을 설계하는 과정을 경험했다.

## Problem
- 표준화 규칙을 실제 데이터 모델에 어떻게 적용해야 하는지, 그리고 실무에서는 어느 수준까지 표준화를 수행하는지 이해하는 데 어려움이 있었다.
- 모델링 과정에서 주문, 배송, 옵션 등 엔티티 간 관계를 설계할 때 적절한 관계와 속성을 결정하는 것이 쉽지 않았다.

## Try
- 데이터 표준화 규칙과 명명 규칙을 다시 정리하여 일관성 있는 모델을 설계하는 연습을 하겠다.
- 다양한 서비스의 ERD를 분석하고 직접 모델링을 반복하며 관계 설계와 정규화에 익숙해지겠다.
