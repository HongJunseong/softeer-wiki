# 2026-07-16 회고

## Keep
- MapReduce를 활용하여 Word Count, Twitter 감성 분석, 영화 평점 평균 계산, Amazon 리뷰 분석 등 다양한 데이터 처리 예제를 직접 구현하며 MapReduce의 동작 방식을 익혔다.
- Mapper와 Reducer의 역할을 이해하고, 여러 데이터셋에 맞게 로직을 수정하며 분산 처리 방식에 익숙해질 수 있었다.

## Problem
- 문제마다 Mapper와 Reducer의 로직이 달라져 데이터를 어떻게 Key-Value 형태로 변환해야 하는지 처음에는 헷갈렸다.
- 입력 데이터 형식에 따라 예외 처리가 필요해 디버깅하는 데 시간이 걸렸다.

## Try
- 다양한 MapReduce 문제를 직접 구현해 보며 Mapper와 Reducer 설계에 익숙해지겠다.
- 데이터 흐름과 Key-Value 변환 과정을 먼저 정리한 뒤 구현하는 습관을 들이겠다.
