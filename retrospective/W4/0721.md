# 2026-07-21 회고

## Keep
- 실제 대용량 데이터를 활용해 지역별, 시간대별 데이터를 분석하며 Spark를 이용한 데이터 탐색 과정을 경험했다.
- 다양한 집계와 시각화를 수행하면서 데이터의 특징을 파악하고, 이를 바탕으로 서비스에 활용할 수 있는 방향을 고민해 보았다.

## Problem
- 분석 결과를 해석하는 과정에서 단순한 수치 확인에 그치지 않고, 객관적인 근거를 바탕으로 인사이트를 도출하는 것이 어려웠다.
- 데이터의 특성과 한계를 충분히 고려하지 못해 의미 있는 분석 방향을 설정하는 데 시간이 많이 소요되었다.

## Try
- 분석을 시작하기 전에 데이터가 제공하는 정보와 한계를 먼저 파악하고, 그에 맞는 분석 목표를 설정하는 습관을 기르겠다.
- 다양한 데이터 분석 사례를 참고하며, 결과를 비즈니스 관점에서 해석하고 설득력 있는 인사이트로 연결하는 연습을 지속하겠다.
