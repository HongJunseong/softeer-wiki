# mtcars 데이터셋 분석

## 1. 기본 데이터 확인

이번 미션에서는 먼저 `mtcars` 데이터셋의 전체 구조를 확인하는 방법을 익혔다.

- `head()`를 사용해 상위 5개 행을 확인했다.
- `tail()`을 사용해 하위 5개 행을 확인했다.
- `shape`를 통해 데이터셋의 행과 열 개수를 확인했다.
- `info()`와 `dtypes`로 각 컬럼의 자료형과 결측 여부를 확인했다.
- `describe()`를 사용해 평균, 표준편차, 최솟값, 최댓값 같은 기본 통계값을 확인했다.

이를 통해 분석을 시작하기 전에 데이터의 크기, 컬럼 구성, 자료형, 기본 분포를 먼저 파악하는 것이 중요하다는 것을 배웠다.

## 2. gear와 transmission 조합 분석

`gear`와 `am` 컬럼을 기준으로 차량이 어떻게 분포되어 있는지 확인했다.

- `unique()`를 사용해 각 컬럼에 어떤 값들이 존재하는지 확인했다.
- `pd.crosstab()`을 사용해 두 컬럼의 조합별 빈도를 표 형태로 확인했다.
- `groupby()`와 `size()`를 사용해 같은 조합별 개수를 구할 수 있었다.

이 과정에서 같은 결과라도 `crosstab`과 `groupby`를 사용해 서로 다른 방식으로 표현할 수 있다는 것을 배웠다.

## 3. 전체 컬럼 히스토그램 시각화

데이터셋에 포함된 여러 컬럼의 분포를 확인하기 위해 히스토그램을 그렸다.

- `subplots()`를 사용해 여러 그래프를 한 화면에 배치했다.
- `enumerate()`로 컬럼을 순회하면서 각 컬럼별 히스토그램을 생성했다.
- `plot(kind='hist')`를 사용해 히스토그램 형태로 데이터를 시각화했다.

이를 통해 숫자형 변수들이 어떤 값의 범위에 많이 분포하는지 직관적으로 확인할 수 있었다.

## 4. Scatter Plot을 통한 관계 확인

두 변수 사이의 관계를 확인하기 위해 산점도를 사용했다.

- `plt.scatter()`를 사용해 두 변수의 관계를 점으로 표현했다.
- `plt.grid()`를 사용해 그래프를 더 읽기 쉽게 만들었다.
- 예를 들어 `cylinder`와 `horsepower`의 관계를 확인하며, 실린더 수가 많을수록 마력이 높아지는 경향이 있는지 살펴볼 수 있었다.

산점도는 두 변수 간의 패턴이나 상관관계를 시각적으로 파악하는 데 유용하다는 것을 배웠다.

## 5. 상관계수 표 확인

마지막으로 변수들 간의 관계를 수치적으로 확인하기 위해 상관계수를 구했다.

- `df.corr()`를 사용해 숫자형 컬럼들 간의 상관계수 표를 만들었다.
- 상관계수는 두 변수의 관계가 양의 방향인지, 음의 방향인지, 또는 약한 관계인지 확인하는 데 도움이 되었다.
- 예를 들어 차량 무게와 연비처럼 서로 반대 방향으로 움직이는 변수들을 파악할 수 있었다.

이번 미션을 통해 데이터 분석의 기본 흐름인 데이터 확인, 조합 분석, 시각화, 상관관계 분석 과정을 전체적으로 경험할 수 있었다.
