# Week 4

4주차에는 Docker 환경에서 Apache Spark Standalone Cluster를 직접 구축하며 Spark 기반의 분산 처리 환경을 구성하는 방법을 학습했다.

Apache Spark를 활용해 NYC TLC Trip Record 데이터를 분석하며 대용량 데이터를 효율적으로 처리하고, 데이터 집계와 분석을 통해 의미 있는 인사이트를 도출하는 과정을 실습했다.

또한 웹 크롤링의 발전 과정과 최신 대규모 크롤링 기술을 학습하며, 대규모 웹 데이터를 효율적으로 수집하고 처리하는 방법에 대해 이해하였다.
