# NYC TLC Trip Record Analysis using Apache Spark

## 1. NYC TLC Trip 데이터 분석 환경 구축

이번 미션에서는 Apache Spark를 활용하여 NYC TLC Trip Record 데이터를 분석할 수 있는 환경을 구성하였다.

- Spark를 이용해 대용량 Trip 데이터를 읽고 처리했다.
- 분산 처리 환경에서 데이터 집계가 가능하도록 구성했다.

Apache Spark를 활용하면 대용량 교통 데이터도 효율적으로 처리할 수 있음을 확인할 수 있었다.

## 2. Trip 데이터 분석 및 집계

NYC TLC Trip 데이터를 이용하여 다양한 집계와 분석을 수행하였다.

- 시간대별, 지역별 Trip 데이터를 집계했다.
- Spark의 DataFrame API를 활용하여 필요한 통계 정보를 계산했다.
- 대용량 데이터를 병렬 처리하여 분석 결과를 도출했다.

Spark의 분산 처리 방식을 활용해 많은 양의 데이터를 빠르게 분석하는 과정을 익힐 수 있었다.

## 3. 분석 결과 확인

집계 결과를 확인하며 데이터의 특징을 분석하였다.

- 지역별 및 시간대별 Trip 분포를 확인했다.
- 분석 결과를 통해 데이터의 패턴과 특성을 파악하였다.

이번 미션을 통해 Apache Spark를 이용한 대용량 데이터 분석 과정과 DataFrame 기반 데이터 처리 방식을 익힐 수 있었다.
