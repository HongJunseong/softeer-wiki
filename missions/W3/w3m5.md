# Average Rating of Movies using MapReduce

## 1. MapReduce 프로그램 작성

이번 미션에서는 Hadoop MapReduce를 이용하여 영화의 평균 평점을 계산하는 프로그램을 작성하였다.

- Mapper에서 영화별 평점 데이터를 출력했다.
- Reducer에서 평점의 합계와 개수를 이용해 평균을 계산했다.

MapReduce를 이용해 데이터를 집계하는 방법을 익힐 수 있었다.

## 2. HDFS 데이터 처리

영화 평점 데이터를 HDFS에 저장하고 작업을 수행하였다.

- 입력 데이터를 HDFS에 업로드했다.
- MapReduce 작업을 실행하여 데이터를 처리했다.

HDFS의 데이터를 기반으로 분산 연산을 수행하는 과정을 이해할 수 있었다.

## 3. 결과 확인

평균 평점 계산 결과를 확인하였다.

- 결과 파일을 HDFS에서 조회했다.
- 영화별 평균 평점이 정상적으로 계산되는 것을 확인했다.

이번 미션을 통해 MapReduce를 활용한 집계 연산과 대용량 데이터 처리 과정을 익힐 수 있었다.
