# Week 3

3주차에는 Docker 환경에서 Hadoop Single-Node와 Multi-Node Cluster를 직접 구축하며 분산 처리 환경을 구성하는 방법을 학습했다.

HDFS를 이용해 데이터를 저장하고 관리하는 방법을 익혔으며, Hadoop MapReduce를 활용하여 대용량 데이터를 분산 처리하는 과정을 실습했다.

또한 Word Count와 감성 분석 MapReduce 프로그램을 작성하고 실행하며 Hadoop의 분산 처리 방식과 데이터 처리 흐름을 이해하였다.
