# Understanding of Hadoop Configuration Files

## 1. Hadoop 설정 파일 이해

이번 미션에서는 Hadoop을 구성하는 주요 설정 파일의 역할을 학습하였다.

- `core-site.xml`
- `hdfs-site.xml`
- `mapred-site.xml`
- `yarn-site.xml`

각 설정 파일이 Hadoop의 서로 다른 기능을 담당한다는 것을 이해할 수 있었다.

## 2. 설정 값 구성

클러스터 환경에 맞게 필요한 설정 값을 직접 수정하였다.

- HDFS 주소와 저장 경로를 설정했다.
- 데이터 복제 수와 MapReduce 실행 환경을 설정했다.
- YARN의 리소스 관리 관련 설정을 구성했다.

설정 파일을 통해 Hadoop 클러스터의 동작 방식을 정의할 수 있다는 것을 배웠다.

## 3. 설정 자동화

반복되는 설정 작업을 자동화하는 방법을 실습하였다.

- Python을 이용해 XML 설정 파일을 생성 및 수정했다.
- 동일한 설정을 여러 노드에 쉽게 적용할 수 있도록 구성했다.

이번 미션을 통해 Hadoop의 주요 Configuration File의 역할을 이해하고, 환경에 맞게 설정 및 관리하는 방법을 익힐 수 있었다.
