# Twitter Sentiment Analysis using MapReduce

## 1. MapReduce 기반 감성 분석

이번 미션에서는 Hadoop MapReduce를 활용하여 Twitter 데이터의 감성 분석을 수행하였다.

- 입력 데이터를 HDFS에 저장하였다.
- Mapper와 Reducer를 이용해 감성 분석 작업을 수행하였다.

MapReduce를 활용해 대용량 데이터를 분산 처리하는 방법을 익힐 수 있었다.

## 2. 데이터 처리

감성 분석을 위해 입력 데이터를 전처리하고 처리하였다.

- Twitter 데이터를 HDFS에서 읽어왔다.
- 감성 분류 결과를 집계하여 출력하였다.

분산 환경에서도 데이터 전처리와 분석을 수행할 수 있다는 점을 배웠다.

## 3. 결과 확인

MapReduce 작업 결과를 확인하고 분석하였다.

- 결과 파일을 HDFS에서 조회하였다.
- 감성 분석 결과가 정상적으로 집계되는 것을 확인하였다.

이번 미션을 통해 Hadoop MapReduce를 이용한 감성 분석의 기본적인 처리 과정과 실행 방법을 익힐 수 있었다.
