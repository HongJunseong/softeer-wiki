# Amazon Product Review using MapReduce

## 1. MapReduce 기반 리뷰 분석

이번 미션에서는 Hadoop MapReduce를 이용하여 Amazon 상품 리뷰 데이터를 분석하였다.

- 리뷰 데이터를 HDFS에 저장하였다.
- Mapper와 Reducer를 이용해 리뷰 정보를 집계하였다.

MapReduce를 활용하여 대용량 리뷰 데이터를 분산 처리하는 방법을 익힐 수 있었다.

## 2. 데이터 집계

리뷰 데이터를 원하는 기준으로 집계하였다.

- 필요한 정보를 추출하여 Key-Value 형태로 출력했다.
- Reducer에서 데이터를 집계하여 결과를 생성했다.

MapReduce를 이용하면 대량의 데이터를 효율적으로 분석할 수 있다는 점을 배웠다.

## 3. 결과 확인

분석 결과를 확인하고 검증하였다.

- 결과 파일을 HDFS에서 조회했다.
- 집계 결과가 정상적으로 출력되는 것을 확인했다.

이번 미션을 통해 Hadoop MapReduce를 활용한 대용량 리뷰 데이터 분석과 집계 과정을 익힐 수 있었다.
