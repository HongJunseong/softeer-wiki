# Word Count using MapReduce

## 1. MapReduce 프로그램 작성

이번 미션에서는 Hadoop의 MapReduce를 이용하여 Word Count 프로그램을 작성하였다.

- Mapper에서 단어를 추출하여 `(단어, 1)` 형태로 출력했다.
- Reducer에서 같은 단어의 개수를 합산하였다.

MapReduce가 데이터를 분산 처리하는 기본 구조를 이해할 수 있었다.

## 2. HDFS 데이터 처리

입력 파일을 HDFS에 저장하고 MapReduce 작업을 실행하였다.

- 입력 데이터를 HDFS에 업로드했다.
- 작업 결과를 HDFS에 저장하고 확인했다.

MapReduce는 HDFS의 데이터를 직접 읽고 결과를 다시 HDFS에 저장한다는 점을 배웠다.

## 3. 작업 실행 및 결과 확인

Word Count 작업을 실행하고 결과를 확인하였다.

- Hadoop Job을 실행하여 전체 데이터를 처리했다.
- 결과 파일을 조회하여 단어별 빈도를 확인했다.

이번 미션을 통해 Hadoop MapReduce의 실행 과정과 분산 데이터 처리의 기본 흐름을 익힐 수 있었다.
