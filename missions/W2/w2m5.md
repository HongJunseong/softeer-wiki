# Sentiment Analysis

## 1. YouTube 댓글 크롤링

이번 미션에서는 Selenium을 활용하여 YouTube 영상의 댓글을 수집하는 방법을 학습했다.

- Selenium으로 YouTube 페이지를 자동으로 제어하였다.
- 스크롤을 반복하여 댓글을 수집하였다.
- 댓글 데이터를 CSV 파일로 저장하여 분석에 활용하였다.

동적 웹 페이지에서 데이터를 자동으로 수집하는 크롤링 과정을 익힐 수 있었다.

## 2. 데이터 전처리

감정분석을 수행하기 전에 댓글 데이터를 전처리하였다.

- HTML 태그와 특수문자를 제거하였다.
- 이모지와 불필요한 공백을 정리하였다.
- 분석에 사용할 수 있도록 데이터를 정제하였다.

텍스트 데이터는 그대로 분석하기보다 전처리를 거쳐야 더 정확한 결과를 얻을 수 있다는 점을 배웠다.

## 3. 감정분석(Sentiment Analysis)

전처리한 댓글을 이용하여 감정분석을 수행하였다.

- 댓글을 긍정과 부정으로 분류하였다.
- 감정분석 결과를 통해 차량에 대한 사용자 반응을 확인하였다.
- 긍정과 부정의 비율을 비교하여 전체적인 여론을 파악하였다.

감정분석을 통해 많은 댓글을 직접 읽지 않아도 사용자들의 전반적인 의견을 빠르게 확인할 수 있다는 점을 이해할 수 있었다.

## 4. WordCloud를 활용한 시각화

감정분석 결과와 함께 댓글의 주요 키워드를 WordCloud로 시각화하였다.

- 자주 등장하는 단어를 추출하였다.
- 불용어를 제거하여 의미 있는 단어만 시각화하였다.
- 단어의 등장 빈도에 따라 크기가 달라지는 WordCloud를 생성하였다.

WordCloud를 통해 댓글에서 어떤 단어가 많이 언급되는지 직관적으로 확인할 수 있었으며, 감정분석 결과와 함께 사용자들의 관심사를 파악하는 데 도움이 되었다.

## 5. 결과 분석

감정분석과 WordCloud를 함께 활용하여 댓글 데이터를 분석하였다.

- 차량별 긍정·부정 반응을 확인하였다.
- 자주 언급되는 키워드를 통해 소비자의 관심 요소를 파악하였다.
- 텍스트 데이터를 정량적 분석과 시각화를 함께 활용하면 사용자 의견을 더욱 효과적으로 이해할 수 있다는 점을 경험하였다.

이번 미션을 통해 YouTube 댓글을 직접 크롤링하고, 데이터를 전처리한 뒤 감정분석과 WordCloud를 활용하여 사용자들의 반응을 분석하는 전체 과정을 경험할 수 있었다.
