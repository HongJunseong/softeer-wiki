# Comment : Predicting Election Results from Twitter Using Machine Learning Algorithms

**Article:** <https://www.researchgate.net/publication/343310126_Predicting_Election_Results_from_Twitter_Using_Machine_Learning_Algorithms>

이 논문을 통해 SNS, 특히 트위터 데이터를 활용하여 선거 결과를 예측할 수 있다는 점이 흥미롭게 느껴졌다. 기존의 여론조사뿐만 아니라 사용자가 직접 작성한 데이터를 감성 분석과 머신러닝으로 분석하여 실제 선거 결과와 유사한 예측을 얻었다는 점이 인상적이었다.

다만 트위터 이용자만을 대상으로 분석하기 때문에 전체 유권자의 의견을 모두 반영하기에는 한계가 있을 것이라고 생각했다. 그럼에도 실시간으로 대량의 데이터를 수집하고 분석할 수 있다는 점에서 선거뿐만 아니라 다양한 사회 현상을 예측하는 분야에도 충분히 활용될 수 있는 연구라고 느꼈다.
