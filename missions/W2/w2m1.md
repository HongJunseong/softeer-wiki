# Multiprocessing Pool

## 1. Pool 생성

이번 미션에서는 `multiprocessing.Pool`을 사용하여 여러 작업을 동시에 처리하는 방법을 학습했다.

- `Pool(processes=2)`를 사용해 워커 프로세스를 생성했다.
- 여러 작업을 동시에 처리하기 위해 Pool의 개념을 이해했다.

Pool을 사용하면 직접 프로세스를 생성하지 않아도 여러 작업을 효율적으로 분배할 수 있다는 것을 배웠다.

## 2. 작업(Task) 정의

여러 개의 작업을 리스트 형태로 정의하였다.

- 작업 이름과 수행 시간을 하나의 튜플로 저장했다.
- 각 작업은 `(이름, 수행시간)` 형태로 구성하였다.

작업 정보를 하나의 리스트로 관리하면 여러 작업을 쉽게 처리할 수 있다는 것을 배웠다.

## 3. 작업 함수 구현

각 작업을 수행하는 함수를 작성하였다.

- 작업 시작 메시지를 출력했다.
- `time.sleep()`을 사용해 작업 수행 시간을 모의 실험하였다.
- 작업 완료 메시지를 출력하였다.

실제 작업 대신 `sleep()`을 사용하여 병렬 처리 동작을 확인할 수 있었다.

## 4. Pool.map() 사용

Pool의 `map()` 함수를 이용하여 모든 작업을 실행하였다.

- `map()`을 사용하여 작업 리스트를 함수에 전달하였다.
- 여러 작업이 워커 프로세스에 자동으로 분배되는 것을 확인하였다.

이번 미션을 통해 `multiprocessing.Pool`을 이용한 병렬 처리의 기본 사용 방법을 익힐 수 있었다.
