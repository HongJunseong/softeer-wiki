# Multiprocessing Process

## 1. Process 생성

이번 미션에서는 `Process` 클래스를 사용하여 새로운 프로세스를 생성하는 방법을 학습했다.

- `Process()`를 사용하여 새로운 프로세스를 생성하였다.
- `target`을 이용해 실행할 함수를 지정하였다.

프로세스는 하나의 독립적인 실행 단위라는 것을 이해할 수 있었다.

## 2. 프로세스 실행

생성한 프로세스를 실행하였다.

- `start()`를 사용하여 프로세스를 시작하였다.
- 여러 프로세스를 동시에 실행하는 방법을 확인하였다.

`start()`를 호출해야 프로세스가 실제로 실행된다는 것을 배웠다.

## 3. 프로세스 종료 대기

프로세스가 모두 종료될 때까지 기다리는 방법을 학습하였다.

- `join()`을 사용하여 프로세스 종료를 기다렸다.
- 모든 작업이 끝난 뒤 다음 코드가 실행되도록 구성하였다.

`join()`을 통해 프로세스 간 실행 순서를 제어할 수 있다는 것을 배웠다.

이번 미션을 통해 `Process`를 생성하고 실행하는 기본적인 방법을 익힐 수 있었다.
