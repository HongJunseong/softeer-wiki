# Multiprocessing Queue

## 1. Queue 생성

이번 미션에서는 `Queue`를 사용하여 프로세스 간 데이터를 전달하는 방법을 학습했다.

- `Queue()`를 생성하여 데이터를 저장하였다.
- Queue가 FIFO(First In First Out) 구조로 동작하는 것을 확인하였다.

Queue를 이용하면 여러 프로세스가 안전하게 데이터를 공유할 수 있다는 것을 배웠다.

## 2. Queue에 데이터 저장

리스트에 있는 데이터를 Queue에 저장하였다.

- `put()`을 사용하여 데이터를 Queue에 추가하였다.
- 데이터를 저장하면서 번호와 값을 함께 출력하였다.

Queue는 입력한 순서대로 데이터가 저장된다는 것을 확인하였다.

## 3. Queue에서 데이터 꺼내기

Queue에 저장된 데이터를 하나씩 꺼내 출력하였다.

- `get()`을 사용하여 데이터를 꺼냈다.
- Queue가 빌 때까지 반복하여 모든 데이터를 출력하였다.

Queue는 먼저 저장한 데이터가 먼저 꺼내지는 FIFO 구조임을 확인할 수 있었다.

이번 미션을 통해 `Queue`를 이용한 데이터 전달과 FIFO 구조를 이해할 수 있었다.
